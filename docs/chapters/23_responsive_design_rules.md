# 23. Responsive Design Rules

### Breakpoints
- **Mobile:** 320px - 767px (4-column layout).
- **Tablet:** 768px - 1023px (8-column layout).
- **Desktop:** 1024px+ (12-column layout).

No horizontal scroll is allowed. Mobile elements must wrap cleanly.
