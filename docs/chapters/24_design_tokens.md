# 24. Design Tokens

All sizes and dimensions are specified under design tokens:
- **Spacing:** xs (4px), sm (8px), md (16px), lg (24px), xl (40px).
- **Radius:** sm (6px), md (12px), lg (20px), full (9999px).
- **Shadows:** glass, card.
