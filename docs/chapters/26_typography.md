# 26. Typography

### Font Families
- **Headings:** Outfit (Outfit Sans)
- **Body Text:** Inter (Inter Variable)

### Typography Scale
- **H1:** 36px (Line height 1.2)
- **H2:** 28px (Line height 1.3)
- **H3:** 24px (Line height 1.4)
- **Body:** 16px (Line height 1.5)
